ARRAY

fill

array1.fill(a,b,c)
array[b]부터 [c]까지 a로 채움
array1.fill(a,b)
array[b]부터 끝까지 a로 채움
array1.fill(a)
배열을 전부 a로 채움

filter, find, findIndex(lastIndexOf)

filter: 조건을 만족하는 모든 값.
const words = ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present'];
const result = words.filter(word => word.length > 6);

find: 조건을 만족하는 첫 번째 값. 없으면 undefined 리턴
const found = array1.find(element => element > 10);

findIndex: 조건을 만족하는 첫 번째 값의 인덱스. 없으면 -1 리턴
lastIndexOf:  조건을 만족하는 마지막 값의 인덱스. 없으면 -1 리턴
const isLargeNumber = array1.findIndex(element => element > 13);

람다(익명함수): function()을 간략하게 쓸 수 있음
let normalFunc = function(x){
	console.log(x);
}
         ↑같음↓
let arrowFunc = (x) => {
	console.log(x);
}

forEach

array의 각 요소들마다 함수 실행
const array1 = ['a', 'b', 'c'];
array1.forEach(element => console.log(element));

pop, push, shift, unshift

pop:배열의 마지막을 없애고 반환.
push:배열의 마지막에 추가하고 배열의 새로운 길이 반환
shift:배열의 첫 번째를 없애고 제거된 요소 반환
unshift:배열의 첫 번째를 없애고 새로운 길이 반환

sort

배열의 요소를 첫 글자를 기준으로 아스키 코드 순으로 나열

include

array.includes('a');
배열이 'a'를 포함하고 있는지 확인하고 true, false 반환
